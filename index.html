What is bootstrap-demo.js?

Easy.

bootstrap-demo.js is a simple (and also unofficial) solution to schooling those pesky end-users we've all heard about.

It allows you to easily create demos and tutorials for your own applications via invocation using javascript or data-attributes.

Easy!

Since it's basically a combination of an overlay and Twitter Bootstrap popovers, you can use any of the default popover functionality.

bootstrap-demo.js also manages demo display through the use of cookies: all you need to do is provide an ID for a demo and you're set!

Easy?!

bootstrap-demo-js allows for complete and total world-domination customization.

You can entertain yourself for hours by customizing:

  • Overlay & Highlight styling
  • Popover positioning
  • And more!

Getting it

Clone the repo, cd to the directory and copy the minified script to wherever you actually need it to be. Standard practice, really.

git clone git://github.com/kalmanolah/bootstrap-demo.js.git
cp bootstrap-demo.js/src/bootstrap-demo.min.js /path/to/my/app/assets/js/

You could also save the good people at Github some bandwidth by doing:

wget https://github.com/kalmanolah/bootstrap-demo.js/raw/master/src/bootstrap-demo.min.js

Using it

There are two ways for you to use bootstrap-demo.js: normal script or the data api.

Normal script:

$('#selector').demo({
	id		: 2,
	placement	: 'top',
	content		: 'This is a tip with <strong>html</strong>',
	title		: 'WARNING <i class="icon-warning-sign"></i>'
});

Data API:

<div 
	data-provide="demo" 
	data-id="3" 
	data-placement="bottom" 
	data-title="ProTip (tm)" 
	data-content="Click me!">
	Content goes here.
</div>

Examples

Here are some examples that demonstrate the full capabilities of bootstrap-demo.js.

Multiple popovers at the same time

$('#home .btn').demo({
	id		: '1-1',
	content		: 'This is the first demo!',
	placement	: 'bottom'
});

$('#home .btn-success').demo({
	id		: '1-2',
	content		: 'This is the second one!',
	placement	: 'top'
});
Run

Demo chaining with callbacks

$('.example-2').click(function(e){
	$('#home .btn').demo({
		id		: '1-1',
		content		: 'Are you ready for some chaining?',
		placement	: 'left',
		callback	: function(e) {
			example21();

			// Don't follow link
			e.preventDefault(true);
			return false;
		}
	});

	// Don't follow link
	e.preventDefault(true);
	return false;
});

function example21(){
	$('#home .btn-success').demo({
		id		: '1-2',
		content		: 'Here goes...',
		placement	: 'top',
		callback	: function(e) {
			example22();

			// Don't follow link
			e.preventDefault(true);
			return false;
		}
	});
}

function example22(){
	$('#about').demo({
		id		: '1-3',
		content		: 'Wham!',
		placement	: 'top'
	});
}
Run

Style customization

$('.example-3').click(function(e){
	$('.example-3').demo({
		id				: '1-1',
		title				: 'Oh boy, I\'m excited.',
		'overlay-style'			:
			'position: fixed;'+
			'top:0;left:0;bottom:0;right:0;'+
			'width:100%;height:100%;'+
			'z-index:1040;'+
			'background:rgba(178,45,45,0.5);',
		'highlight-style'		:
			'z-index:1050;'+
			'background:rgba(255,255,255,1);'
	});

	// Don't follow link
	e.preventDefault(true);
	return false;
});
Run
